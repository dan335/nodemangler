A node-based image editor and procedural texture generator.

Similar to Substance Designer and Material Maker, but open source, free, and not built on top of a game engine. Written entirely in Rust. Build graphs in the desktop app, or run them headless from the command line — both share the same JSON graph format.

the graph editor
free & open source· windows / mac / linux· gui + cli· 100% rust
Procedural

Noise, patterns, shapes, and generators for building textures from nothing — fractal, cellular, grunge, and simulation-based.

PBR materials

Normal, AO, curvature, and bevel from height maps, a live 3D material preview, and channel-packed export presets for Godot, Unity, and Unreal.

Image editing

Adjustments, filters, morphology, blurs, and transforms in a floating-point pipeline — images stay f32 from input to output.

Color

Construct, convert, and analyze colors across every major color space, with harmony and blend-mode nodes.

Resolution independent

Spatial parameters scale with output size: design at 512px, render at 4K, same result — don't touch the numbers.

Scriptable

The CLI runs graphs headless and is easy for scripts and LLMs to drive. Graphs round-trip freely between GUI and CLI, and pipelines package into reusable subgraph nodes.

git clone https://github.com/dan335/nodemangler
cd nodemangler/app
cargo run -p mangler_gui    # desktop app
cargo run -p mangler_cli    # headless cli
