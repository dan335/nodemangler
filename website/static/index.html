A node-based image editor and procedural texture generator.

Similar to Substance Designer and Material Maker, but open source, free, and not built on top of a game engine. Written entirely in Rust. Build graphs in the desktop app, or run them headless from the command line — both share the same JSON graph format.

the graph editor
380 nodes· 45 noise generators· 14 color spaces· 16 file formats· gui + cli
Procedural

Noise, patterns, shapes, and PBR nodes for building textures from nothing. Package a pipeline into a single reusable subgraph node.

Image editing

Adjustments, filters, morphology, and transforms in a floating-point pipeline — images stay f32 from input to output.

Color

Fourteen color spaces with conversion between them, plus harmony, analysis, and manipulation nodes.

Scriptable

The CLI runs graphs headless and is easy for scripts and LLMs to drive. Graphs round-trip freely between GUI and CLI.
